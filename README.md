## how to use(Example)
### build
```sh
forge build -C ./Counter
```

### test
```sh
forge test -C ./Counter
```

### deploy
```sh
cp .env.example .env

# add the keys in .env

source .env

forge script ./Counter/script/Counter.s.sol:CounterScript --fork-url $GOERLI_RPC_URL --broadcast -vvv
```
